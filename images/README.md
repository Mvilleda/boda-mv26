# Images Folder

Place your wedding photos here. The website expects the following images:

## Required Images:

1. **hero-bg.jpg** - Background image for the hero section (large, high quality recommended)
2. **couple-photo.jpg** - Main photo of the couple for the welcome section
3. **photo1.jpg** through **photo6.jpg** - Six photos for the gallery section

## Image Guidelines:

- **Format:** JPG, JPEG, or PNG
- **Resolution:** At least 1920x1080 for hero-bg.jpg, 800x1000 for couple-photo.jpg
- **Size:** Optimize images to keep file sizes reasonable (under 2MB each if possible)
- **Orientation:** 
  - hero-bg.jpg: Landscape
  - couple-photo.jpg: Portrait or landscape
  - Gallery photos: Any orientation works

## How to Add Images:

1. Download your images from Canva or your photo source
2. Rename them according to the list above
3. Place them in this `images/` folder
4. The website will automatically display them

## Optional:

You can add more photos to the gallery by:
1. Adding more image files (photo7.jpg, photo8.jpg, etc.)
2. Adding corresponding `<div class="gallery-item">` entries in index.html
