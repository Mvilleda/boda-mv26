# Fonts Folder

Para usar la fuente "Hello Paris" en tu sitio web, necesitas agregar el archivo de la fuente aquí.

## Cómo obtener la fuente Hello Paris:

### Opción 1: Desde Canva (si tienes Canva Pro)
1. No es posible descargar fuentes directamente desde Canva

### Opción 2: Comprar/Descargar la fuente
1. Busca "Hello Paris font" en sitios como:
   - Creative Market
   - MyFonts
   - FontSpring
   - Dafont (si hay versión gratuita)

2. Descarga el archivo de fuente (generalmente viene en formato .ttf, .otf, .woff, o .woff2)

3. Coloca el archivo aquí en la carpeta `fonts/` con el nombre:
   - `HelloParis.ttf` (preferido)
   - o `HelloParis.woff`
   - o `HelloParis.woff2`

### Opción 3: Usar una fuente alternativa similar
Si no puedes obtener "Hello Paris", fuentes similares gratuitas son:
- **Allura** (Google Fonts)
- **Alex Brush** (Google Fonts)
- **Parisienne** (Google Fonts)
- **Great Vibes** (Google Fonts)

Para usar una alternativa de Google Fonts, dime cuál prefieres y actualizaré el código.
